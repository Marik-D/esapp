# Examples

To run the dev server with the examples build the package (optionally in watch mode).

```bash
yarn build --watch
```

## App Server

To run the app server.

```bash
cd examples
../bin/main.js dev
```

## Book

To run the storybook:

```bash
cd examples
../bin/main.js book
```

To build the storybook:

```bash
cd examples
../bin/main.js book --build
```
